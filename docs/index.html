Welcome to the MapleStory 2 Skill Charts

MapleStory 2 Skill Charts is a set of web components which, with very little effort, let you add an interactive skill chart to your website or blog. This way your viewers can interact with the chart and see exactly what bonuses your chosen skill gives. So much better than just a screenshot!

How do I put this one my site or blog?

Select a class and the skills you want to show and then copy the code below into your site or blog.

What you see below is an editor. When you use this on your site or blog the skills will no longer be editable.

Archer Assassin Berserker Heavy Gunner Knight Priest Runeblade Thief Wizard

<script src="https://unpkg.com/maplestory-skills@^0/dist/maplestory-skills.js"></script>

FAQ

The skill charts are too big!
You can make them smaller with transform. 
ms-archer, ms-assassin, ms-berserker, ms-heavy-gunner,
ms-knight, ms-priest, ms-runeblade, ms-thief, ms-wizard
{
	transform: scale(0.8);         /* adapt margin below if you change this */
	transform-origin: top center;  /* or "top left" if you don't wanna center it */
	margin-bottom: -20%;           /* to remove empty space at the bottom, or */
	margin-right: -20%;            /* if you use "top left" for transform-origin */
}
